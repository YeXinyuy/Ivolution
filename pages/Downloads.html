Binaries and Code

Here is what you need to install Ivolution :

  • For Windows users, the last executable

  • For Linux and Mac users, the source package

  • Complete source code of the last version (updated each time I commit)

Older versions are also available here.

If you want to always have the last version available, you can also choose to directly clone the git repository of the project (and I'll gladly accept your pull requests ! ;))

$ git clone https://github.com/jlengrand/Ivolution.git

Simply enter the project and run ./Ivolutioner in command line should get you started :).
